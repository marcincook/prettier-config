# `marcincook/prettier-config`

> Marcincook [Prettier](https://prettier.io) config.

## Usage

**Install**:

```bash
yarn add --dev marcincook/prettier-config
npm install --dev marcincook/prettier-config
```

**Edit `package.json`**:

```jsonc
{
  // ...
  "prettier": "marcincook/prettier-config"
}
```
